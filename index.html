諮詢需要提供哪些資料？ +
依服務類型而定。命理諮詢（姓名學、紫微斗數）需提供姓名、出生年月日與時辰，越精確越好；風水服務需提供建物地址與等比例平面圖。各服務項目中有詳細列出所需資料。
諮詢如何進行？可以面對面嗎？ +
線上服務以 LINE 視訊或語音進行，方便全台灣及海外客戶預約。實地陽宅堪輿可現場勘察，當場解疑形煞影響，報告完成後另安排線上說明。詳細安排請透過 LINE 聯絡。
報告多久可以拿到？ +
確認付款後，一般排程為 5–7 個工作天完成報告，並安排線上說明。旺季或特殊節日前後可能稍長，會提前告知。
可以諮詢哪些主題？ +
常見主題包含：購屋前陽宅鑑定、居家或辦公室風水調整、新生兒命名、成人改名、命盤解析、事業財運分析、感情婚姻、創業時機等。歡迎直接告知最想了解的面向，我會建議最適合的服務項目。
台灣客戶付款方式有哪些？ +
接受銀行轉帳、LINE Pay。確認服務項目後，會於官方 LINE 提供匯款資訊，完成付款後即開始服務。
海外客戶如何付款？ +
海外客戶可透過 PayPal 付款。確認服務項目後，會於官方 LINE 提供付款連結，完成付款後即開始排程。
